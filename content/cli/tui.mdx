# `cortex tui`

Launch the Terminal User Interface (TUI) for Cortex, providing a rich interactive terminal experience with keyboard navigation and real-time updates.

## Usage

```bash
cortex tui
```

## Overview

The TUI provides a keyboard-driven terminal interface for interacting with Cortex. It offers visual navigation for sessions, memory, jobs, agents, and system status with real-time updates.

The TUI is built on the `src/tui/` module and provides a rich terminal experience with:
- Keyboard shortcuts for navigation
- Real-time status updates
- Session browsing and management
- System health monitoring
- Visual configuration editing

## Examples

```bash
# Launch the terminal UI
cortex tui
```
